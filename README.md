# PHP Personal Blog

A simple personal blog web app built with HTML, CSS, and PHP that used to write and publish articles. This project is inspired by [roadmap.sh](https://roadmap.sh/projects/personal-blog).

## Features

- **View All Articles**: Display a list of all existing articles with their title and publishing date.
- **Admin Login**: A basic authentication for the admin section to be able to manage articles.
- **Add a new article**: Create a new article with a title, publishing date, and content.
- **View an Article**: Display an article with theri title, publishing date, and content.
- **Update an Article**: Modify an existing article.
- **Delete a Task**: Remove an existing article.

## Prerequisites

To run this CLI tool, you’ll need:

- **PHP**: Version 7.4 or newer

## How to install

1. Clone the repository

   ```shell script
   git clone https://github.com/krisnaajiep/php-personal-blog.git
   ```

2. Change the current working directory

   ```shell script
   cd path/php-personal-blog
   ```

3. Run the PHP Bult-in server

   ```shell script
   php -S localhost:8000
   ```

4. Access the URL
   ```shell script
   http://localhost:8000
   ```

## Authentication

Authentication is required for the admin section to be able to access the dashboard page and manage articles using the following credentials-

```
Username: admin
Password: password
```
